Commands

Parameters - Required parameters. [params] - Optional parameters. @user - Takes a user @mention. @role - Takes a role @mention. #channel - Takes a channel #mention. Info Commands help - Displays the help message. server - Get the link to join the official ChewieBot Discord server. invite - Get the link to add ChewieBot to your server. serverinfo - Get info about the server you are in. avatar [@user] - Get the avatar of a user. (Leave out the mention for your own avatar.) Fun Commands poop [@user] - Poop on a user. fart [@user] - Fart at a user. arrest <@user> - Arrest a user. punch [@user] - Punch a user. Utility Commands ping - Check if the bot is responding. subscribe - Subscribe to ChewieBot announcements. unsubscribe - Unsubscribe to ChewieBot announcements. randomnumber - Generate a random number between the min and max. dice - Roll a six-sided dice. coinflip - Flip a coin. bug - Report a bug in the bot. suggest - Suggest a feature. Bot Developer Commands userid [@user] - Get the ID of a user. (Leave out the mention for your own.) channelid [#channel] - Get a channel ID. (Leave out the channel for current channel ID) serverid - Get the ID of the server you are in. roleid <@role> - Get a role ID. (Only works on roles you can mention. Do not cause mention spam.) resources - Get bot development resources. Moderation Commands Note: The reason in mod logs is a bit broken with users with nicknames. kick <@user> - Kicks a user with a reason in mod logs. ban <@user> - Bans a user with a reason in mod logs. warn <@user> - Warns a user with a reason in mod logs. Channels #logs - Logs join/leave, emoji added/removed, role added/removed, etc. #mod-logs - Logs kicks, bans, and warns.
